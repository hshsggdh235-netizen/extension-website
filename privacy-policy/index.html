Learning Assistant Extension

Learning Assistant is a Chrome Extension designed to assist users while studying on online learning platforms.

Features

  • Analyze questions displayed on the current webpage.
  • Generate suggested answers using AI.
  • Secure user authentication.
  • Cloud synchronization of user subscriptions.
  • Fast and secure communication with our backend server.

Security

All communications between the extension and our backend services are encrypted using HTTPS.

Permissions

The extension only requests permissions that are necessary for its intended functionality.
